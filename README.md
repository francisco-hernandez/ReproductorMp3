# Reproductor MP3

Un reproductor de audio MP3 con interfaz moderna y diseño pulido, desarrollado como proyecto personal de ejemplo.

## 🎯 Descripción

Este proyecto es un reproductor de música web con una interfaz de usuario elegante que implementa principios de diseño modernos. Cuenta con controles de reproducción intuitivos, barra de progreso interactiva y un diseño glassmorphism atractivo.

## ✨ Características

- **Interfaz Moderna**: Diseño glassmorphism con efectos de desenfoque y transparencia
- **Controles Intuitivos**: Reproducir/Pausar, Anterior, Siguiente
- **Barra de Progreso Interactiva**: Permite navegar por la pista de audio
- **Diseño Responsivo**: Se adapta a diferentes tamaños de pantalla
- **Animaciones Fluidas**: Transiciones suaves y feedback táctil en botones
- **Accesibilidad**: Soporte para navegación por teclado y etiquetas ARIA

## 🛠️ Tecnologías

- **HTML5**: Estructura semántica
- **CSS3**: Estilos con animaciones y efectos modernos
- **JavaScript**: Lógica de reproducción de audio
- **Bootstrap 5**: Framework CSS para componentes base
- **Bootstrap Icons**: Iconos vectoriales

## 📦 Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/francisco-hernandez/ReproductorMp3.git
```

2. Navega al directorio del proyecto:
```bash
cd ReproductorMp3
```

3. Abre el archivo `index.html` en tu navegador web

## 🚀 Uso

Para usar el reproductor:
1. Reemplaza `TU_ARCHIVO_DE_AUDIO.mp3` en el archivo `index.html` con la ruta de tu archivo de audio
2. Abre `index.html` en tu navegador
3. Usa los controles para reproducir, pausar y navegar por la pista
